welke api calls moeten er allemaal zijn?

usertabel: 
1. je moet een user kunnen toevoegen
2. je moet een user kunnen aanpassen
3. je moet een user kunnen verwijderen

tasklisttable:
1. je moet een nieuwe tasklist kunnen maken
2. je moet de tasklist kunnen aanpassen
3. je moet de eigenaar van de tasklist kunnen wijzigen
4. je moet de task list kunnen verwijderen
5. je moet members kunnen toewijzen aan de tasklsit

tasktable: 
1. je moet een nieuwe task kunnen creeeren 
2. je moet de task kunnen toewijzen aan verschillende medewerkers
3. je moet de task kunnen aanpassen
4. je moet de task status kunnen wijzeigen
5. je moet de task kunnen verwijderen
